# Py-Quine
### A Python [Quine](https://en.wikipedia.org/wiki/Quine_(computing))
## What exactly is Py-Quine?
This project is a [Quine](https://en.wikipedia.org/wiki/Quine_(computing)) written in the Python programming language. This means writes its own source code when executed.
## How to run Py-Quine
To execute the project simply fork it to a local drive and execute it with the following command:
```
python3 quine.py
```
The quine will proceed to write its own source code to a file called `quine_cpy.py` in the same directory.
